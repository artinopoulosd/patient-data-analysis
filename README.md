# Patient Data Analysis

A beginner Python project for analyzing fictional patient vital signs using Pandas.

## Project Purpose

The purpose of this project is to practice basic Python and data analysis concepts using a simple healthcare dataset.

The project analyzes fictional patient data including:

- Age
- Gender
- Heart rate
- Systolic blood pressure
- Diastolic blood pressure
- Length of stay

## Technologies

- Python
- Pandas
- CSV
- GitHub

## Analysis

The Python script calculates:

- Average patient age
- Average heart rate
- Average length of stay

It also filters patients with a heart rate above 100 beats per minute.

## Dataset

The dataset contains fictional patient data created for educational purposes.

No real patient or personal health information is used.
